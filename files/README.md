# 극지방의 아이들 — 생태계 시뮬레이션 (동적 지형 + 지구 온난화 버전)

8종(북극곰·북극여우·펭귄·바다표범·범고래·북극대구·크릴·순록)이 빙판과 바다에서
사냥·도망·번식·사망하며 살아가는 극지방 생태계 시뮬레이션이다.

- **온난화가 없으면**: 빙하가 얼고 녹기를 반복하며 비율이 ~50% 에서 일정하게
  유지되고(북극처럼), 8종이 수천 턴 동안 안정적으로 공존한다.
- **AI 사용으로 인한 지구 온난화가 진행되면**: 온도가 계속 올라 **빙하가 녹고
  수온이 상승**, 서식지 상실과 열 스트레스로 위 영양단계부터 자연스럽게 붕괴한다.
  **크릴을 제외한 모든 생물이 사라지면 '멸망'** 으로 본다.

## 실행

```bash
pip install pygame          # 시각화용 (헤드리스 검증은 불필요)
python main.py              # pygame 실시간 시각화 (동적 빙하 격자 + 기후 패널)
python main.py --warming 3000    # 온난화 ON: 붕괴→멸망 시나리오(헤드리스)
python main.py --verify          # 온난화 OFF: 생태 + 빙하비율 안정성 검증
python main.py --headless 2000   # 기본(온난화 ON) 헤드리스
```

| 키 | 기능 |
|----|------|
| `ESC` | 종료 | `Space` | 일시정지 |
| `F` | 4배속 | `M` | 온난화 +3℃ (가속 시연) |

## 지형·기후 시스템 (terrain.py)

빙하를 '고정된 절반'이 아니라 **온도에 따라 얼고 녹는 동적 격자**로 만들었다.

- **공기 온도** = 위도 그라데이션(위=극지방=추움) + 계절 + 칸별 노이즈 + 온난화
  → 칸의 공기온도가 `T_FREEZE(-1.8℃)` 미만이면 결빙, `T_MELT(0.2℃)` 초과면 융해
  (이력현상 band 로 안정). **온난화 0이면 빙하 비율이 ~0.5 에서 계절에 따라
  출렁이며 유지**(검증: 평균 0.51, 범위 0.26~0.76).
- **수온** = 차가운 바다 기본값 + 약한 계절 + 온난화. 온난화 전에는 차가워
  냉수성 종이 안전하고, 온난화가 진행되면 종의 내성을 넘어 열 스트레스가 된다.

공기온도(빙하·육상동물)와 수온(해양동물)을 분리한 이유: 북극 바다는 어디서나
차갑지만 빙하의 결빙/융해는 표면 공기온도가 결정하기 때문이다. 이렇게 해야
'온난화 없을 때 빙하 ~50% 유지 + 해양종 안전'이 동시에 성립한다.

## 지구 온난화 → 멸망 메커니즘

AI 지구 온난화는 **'온도를 계속 올리는 기작'** 만 한다(즉발 전멸 X). 온도가 오르면:

1. **빙하 융해** → 빙하 동물(북극곰·여우·순록)이 갈 곳을 잃어(`ICE_LOSS_DMG`) 죽는다.
2. **수온 상승** → 각 종이 내성 온도(`TEMP_TOLERANCE`)를 넘으면 열 스트레스로 죽는다.
   냉수성이 강한 순서로 먼저 죽는다: 북극대구(4℃) → 펭귄(5℃) → 물범(7℃) → 범고래(8℃).
3. 먹이사슬이 위에서부터 끊기며 **크릴만 남고, 그 시점이 멸망**이다.
   (크릴은 1차 생산 기준종으로 두어 멸망 판정의 기준으로 삼는다.)

검증(`--warming`): 약 1,800턴까지 8종이 건강히 유지되다가 임계점을 지나
**대구·순록·여우 → 펭귄 → 곰 → 물범 → 범고래** 순으로 붕괴, ~2,300턴에 멸망.
`warming_chart.png` 가 빙하 비율·수온·개체수의 연동 붕괴를 보여준다.

## 생태계가 무너지지 않는 이유 (온난화가 없을 때)

이전 버전이 100턴 안에 '전멸 + 크릴 폭증'으로 끝나던 문제를 다음으로 해결했다.

| 장치 | 막아주는 문제 |
|------|---------------|
| 크릴 바이오매스 로지스틱 + 상한 하드클램프 + 바닥 floor | 크릴 폭증 / 토대 소멸 |
| 모든 종 밀도의존(로지스틱) 번식 (∝ 1−N/K) | 폭증·전멸 |
| 배부른 포식자는 휴식(포식압 = 먹이 생산량) | 과포식 붕괴 |
| 저밀도 은신처(Holling III) | 희소종이 0으로 내몰림 |
| 이끼·사체 에너지 순환, 계절 이동 rescue | 자원 단절 / 우연한 전멸 |

`population_chart.png` — 온난화 OFF, 2500턴 동안 8종 유지 + 빙하 비율 ~0.5 일정.

## 주요 속성

- `hunger` 0=배부름~100=굶주림 / `TEMP_TOLERANCE` 열 스트레스 임계 온도
- `ICE_LOSS_DMG` 빙하 동물이 녹은 바다 위일 때 피해 / `CARRYING_CAPACITY(K)` 환경수용력
- 온난화 강도는 `Simulation.warming_rate` 로 조절(0이면 영구 안정).

## 생성형 AI 활용

계획서의 'AI 활용 계획'에 따라 클래스 설계, 행동·개체수 동역학(로지스틱·포화형
포식), 동적 지형/기후 모델 설계에 Claude를 활용했으며, 모든 코드는 담당 조원이
이해한 뒤 직접 검토·수정했다.
